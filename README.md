# Project 11: FinGuard AML Intelligence 🛡️

A professional-grade Anti-Money Laundering (AML) system designed to detect 
suspicious financial activity in e-commerce transactions using 
Anomaly Detection and XGBoost.

### Key Features:
- **Real-Time Monitoring:** Identifying high-risk transfers instantly.
- **Advanced Feature Engineering:** Tracking "Mule" account patterns.
- **Professional Dashboard:** A clean, minimalist Streamlit interface.

*Note: The dataset used (PaySim) is ignored due to size. To run locally, 
download the data from Kaggle and place it in the `/data` folder.*
